XML 예약문자 
<, >, &는 XML tag 표시와 entity를 표시하는 XML 예약문자로, XML 문서에 그대로 사용할 수 없다.
 
< (less-than sign) &lt; 
> (greater-than sign) &gt; 
& (ampersand) &amp;

화학기호 
화학기호는 윗첨자나 아랫첨자를 지정하지 않고 그대로 입력한다.
 
K+ K+ 
Cl- Cl- 
Mg2+ Mg2+ 
CO2 CO2 
H2O H2O 


수학기호 
수학기호는 윗첨자나 아랫첨자를 괄호 "( )" 안에 넣어서 입력한다.
 
102 10(2) 
10-2 10(-2) 
height2.239 height(2.239)
