스킬 활성 타입
0,1로 구분 0 : 액티브 1 : 패시브


적용 대상
본인 - 스킬 사용시 자신에게 자동 시전
파티 멤버 - 스킬 사용시 사용할 멤버 선택(멤버가 범위안에 없으면 시전 불가)
파티 전원 - 사용시 범위안에 파티 전원에게 시전
적 - 지정스킬이 아니면 범위안에 적에게 시전


스킬 타입
즉시 - 사용 즉시 적용
주기적 - 주기 시간을 체크 하여 주기 시간 마다 효과 적용
지속적 - 지속 시간을 체크하여 해당 시간 만큼 효과 적용


캐스팅 타입 
즉시 - 사용 즉시 적용
캐스팅 - 스킬 시전 시간 체크후 시전 시간이 지난후 사용
차징 - 차징 조건 체크(시간)후 조건 만족시 차징 효과 적용
       예외처리 - 캐스팅이 있을 경우 캐스팅이 끝나고 차징 조건 체크
위치지정 - 위치를 지정할수 있는 범위가 출력되며 방향키를 사용하여 범위를 이동할수 있으며
  	   특정 조작(ex-스킬키, 공격키)으로 범위안에 스킬을 사용한다.
대상지정 - 사용 시 범위안에 대상이 출력이 되며, 특정키(ex-1,2,3 숫자키, 십자키)를 사용하여 사용한다.


스킬 시전 시간
초로 구분이 되며, 스킬 버튼을 눌렀을 경우 시전바가 채워지며 모두 채워지면 스킬이 사용된다.
시전중에 피격을 당하게 되면 스킬은 취소가 된다.
차징의 경우 시전바가 모두 채워진 후부터 차징 시간이 체크된다.


차징 최대 시간
초로 구분되며, 차징 최대 시간동안 차징키를 누르고 있어야 조건 만족
차징 중간에 차징를 중단하면 차징 효과가 적용이 되지 않는다.


차징 데미지 증가력
차징 조건 만족시 증가하는 데미지


스킬 재사용 시간
스킬 키를 눌러 스킬을 사용한 직후부터 계산한다.
재사용 시간이 끝나지 않았는데 스킬 키를 누르면 스킬이 사용되지 않는다.


스킬 사정거리
m단위를 사용하며, 스킬의 사정거리를 의미한다.


스킬 공격성격
스킬에 피격되면 해당 공격 성격의 효과가 난다


공격 성격 수치
적용되는 공격 성격에 추가적인 수치가 적용된다.


스킬 데미지 타입
물리, 마법으로 구분되며, 데미지타입에 선택에 따라 아바타의 물리,마법공격력를 기준으로 계산된다.


데미지 랜덤, 고정
ex)물리 공격력 110~130%(데미지 고정) + 300(고정 데미지)


스킬 명중률, 크리티컬 확률
스킬 사용시 아바타의 명중률, 크리티컬 확률과 스킬의 명중률, 크리티컬 확률을 더해서 적용
ex)아바타 크리티컬 확률 5%이고 스킬 크리티컬 확률 4% 이면 총 9%의 크리티컬 확률이 적용된다.


스킬 효과 범위
m단위를 사용하며, 스킬의 효과가 적용되는 범위를 의미한다.


상태이상
상태이상 테이블에서 상태이상 id와 적용 확률을 체크해 적용한다.


파라메터
파라메터 적용 확률 체크 -> 파라메터 타입,수치 체크 -> 효과 적용 시간 만큼 적용


최대 다든 히트수
다단 히트 스킬의 경우 최대 몇대까지 다단 히트가 가능한지 체크


다단 히트 주기 시간
다단 히트 주기 시간마다 히트가 적용된다.
 

스킬 테이블 추가할 부분
데미지 타입
현재 물리, 마법으로만 구분되어 있음. 물리와 마법 두가지 데미지를 한번에 주는 캐릭터의 경우를 위해
물리 + 마법의 데미지 타입 추가 필요


상태이상 타입 수정
상태이상 ID와 확률만 남기고 지운다.


데미지 랜덤 -> 변환 데미지로 수정


효과 확률은 따로 없음.
